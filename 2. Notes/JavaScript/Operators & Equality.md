# Operators & Equality

## Arithmetic Operators
* Arithmetic operators are used to perform mathematical operations between numeric operands
* Simple operators are;
	* `+`
	* `*`
	* `-`
	* `/`
```JavaScript
const a = 5;
const b = 10;
let result = 0;
// Addition

result = a + b;
console.log(result);
```
